# n8n Workflows Collection

This repository contains a collection of workflows built with n8n.

These workflows are designed for self-hosted environments and focus on automation, monitoring, and integration use cases.

---

## Purpose

The goal of this repository is to:

- Version control n8n workflows
- Provide reusable automation templates
- Showcase structured workflow design
- Demonstrate production-oriented automation practices

---

## Included Workflows

### Centralized Error Monitoring

A lightweight monitoring workflow that captures execution errors and sends structured alerts via Telegram.

Designed to improve visibility and reliability of automated processes.

---

## Environment

These workflows are built for:

- Self-hosted n8n
- Docker-based deployments
- VPS environments

---

## Notes

- Credentials are not included in the workflow files.
- Users must configure their own API credentials before activating workflows.
- Workflows are provided as templates and can be adapted to different environments.
